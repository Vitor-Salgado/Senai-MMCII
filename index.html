Componentes de uma placa mãe


É uma peça central responsável por conectar e interligar todos os componentes (processador com memória RAM, disco rígido, placa gráfica). A placa une todas as partes do sistema numa só rede de fios, porque dispõe de caminhos que permitem a troca de informação entre processadores, memórias, placas e etc.

HD

Memória de armazenamento de longa duração, realiza um armazenar localmente de arquivos, programas, jogos e outros tipos de conteúdos. Todas as informações gravadas em um HD são lidas ou escritas por meio de um processo magnético

Memória RAM

A função da memória RAM é guardar temporariamente toda a informação que o computador precisa, seja para aquele momento ou para um futuro próximo

Processador

Tem a função de acelerar, endereçar, resolver ou preparar dados, dependendo da aplicação. Basicamente, um processador é uma poderosa máquina de calcular

Placa de Video

É a responsável por realizar a conexão entre os componentes do computador em si e o monitor, sendo responsável por tudo que é exibido na tela para o usuário

Placa Mãe

É uma peça central responsável por conectar e interligar todos os componentes (processador com memória RAM, disco rígido, placa gráfica). A placa une todas as partes do sistema numa só rede de fios, porque dispõe de caminhos que permitem a troca de informação entre processadores, memórias, placas e etc

Fonte de energia

Transformar a corrente elétrica da rede, que chega em uma voltagem alta, para uma voltagem menor, que é a indicada para o hardware do computador.

Construindo um pendrive bootável

Para a Criação do pendrive bootável Primeiro precisamos de um pendrive de no minimo 8GB de espaço

Posterior a isso podemos baixar a imagem do windos diretamente do site da Microsoft através desse link https://www.microsoft.com/pt-br/software-download/windows10



Feito o download direto no pendrive ele esta pronto para ser usado para fazer a formatação do windows

Formatando o Computador

A formatação do windos pode ser realizada de duas formas uma através do proprio sistema e outra através de uma midia externa como o pendrive bootável

Pelo proprio sistema

Iniciamos a formatação pesquisando pelo sistema de restaurar o pc



Após isso inicia a restauração através do próprio sistema




Para fazer por esse modo e necessário uma conexão com a internet pois o windows vai ser baixado nesse processo

Pelo Pendrive

Para iniciar por esse modo primeiro você precisa ter um pendrive bootável que foi explicado anteriormente

Em posse desse pendrive você pode iniciar o processo entrando no boot do sistema clicando na tecla del durante o processo de inicialização



Feito isso abrira o menu de SETUP




Após abertura do SETUP selecione a aba de boot




Agora coloque a entrada usb onde esta plugado o pendrive como a primeira na lista na ordem de executável,apos isso salve a alteração e reinicie a maquina que o processo de boot se iniciara
